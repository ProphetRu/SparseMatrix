Detailed Description

template<typename T, T Default>
class SparseMatrix< T, Default >

Template class implementing sparse matrices.

Each element is represented as a proxy object that allows accessing elements at specific indices.

Template Parameters
TType of values stored in the matrix.
DefaultValue used when there are no entries at certain indices (default value).
